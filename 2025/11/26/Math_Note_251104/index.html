1104第一次测试错题

函数连续性问题

        对于连续函数f(x)和g(x)的和F(x) = f(x) + g(x),F(x)必定不连续

因为，假定F(x)连续，则有：

g(x) = F(x) − f(x)

则与题意矛盾，必不成立

关于间断点

        我们以一个例题来讲明白间断点

点x=1是函数$y=\frac{x-1}{x^2+2x-3}sin\frac{1}{x}$的(    )

A、可去间断点

B、跳跃间断点

C、无穷间断点

D、震荡间断点

        我们对原式化简，得到：

$$ \begin{document} y=\frac{1}{x+3} \end{document} $$

        (x − 1)上下可消去，则x = 1为函数的可去间断点

        (x + 3)上下不可消去，且x → 3时y ≠ 0，则x = −3为函数的无穷间断点

        当x → 0时，$\frac{1}{x}\rightarrow\infin$，此时$sin\frac{1}{x}$震荡，故x = 0为函数的震荡间断点
