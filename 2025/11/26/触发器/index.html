触发器

一、非钟控触发器

RS触发器

上图中，$S$、$R$均为低电平有效，$R$表示复位，$S$表示置位，基本RS触发器的特征方程为：
$$
\begin{cases}
Q^{n+1}=S_{D}+{\overline{R_{D} } } Q^{n}\
{\overline{S_{D} } }+{\overline{R_{D} } }=1
\end{cases}
$$
其中，${\overline{S_{D} } }+{\overline{R_{D} } }=1$是约束条件，当该条件不满足时，基本RS触发器出现竞争状态，输出不定

基本RS触发器的应用电路是三人抢答电路

相关知识点：状态转移真值表、状态转移图、激励表

二、钟控触发器

1. 钟控RS触发器

钟控RS触发器在基本RS触发器的基础上引入了两个与非门作为时钟控制电路，其直接输入为：
$$
{\overline{S_{D} } }={\overline{S{\cdot}CP} },{\overline{R_{D} } }={\overline{R{\cdot}CP} }
$$
当$CP=0$时，触发器状态维持不变，当$CP=1$时，钟控RS触发器的状态方程为：
$$
\begin{cases}
{Q^{n+1} }=S+{\overline{R}}Q^{n}\
RS=0
\end{cases}
$$
其中，$RS=0$是约束条件，当该条件不满足时，钟控RS触发器出现竞争状态，输出不定

钟控RS触发器的状态转移真值表与基本RS触发器的状态转移真值表相同

2.1. 钟控D触发器

D触发器将钟控RS触发器中的$R$与${\overline{S_{D}}}$连接，只保留了一个$Data$输入端，常用作异步计数器

其直接输入为：
$$

{\overline{S_{D} } }={\overline{D{\cdot}CP} }, {\overline{R_{D} } }={\overline{{\overline{S_{D} } } {\cdot}CP} }

$$
当$CP=0$时，${\overline{S_{D}}}=1$，${\overline{R_{D}}}=1$，触发器状态维持不变

当$CP=1$时，D触发器的特征方程为：
$$
Q_{n+1}=D
$$
D触发器的约束条件自动满足

在时钟作用下，D触发器的次态$Q_{n+1}$始终与$D$相同，因此，D触发器也被称为延迟触发器或D型锁存器

D触发器可以实现下文JK触发器(需要增加逻辑门)和T触发器的所有功能，读者可以尝试改变D触发器使其实现其他触发器的功能

相关应用：74373（8位D型锁存器），基于D触发器的模8加法计数器（8分频器）

3.1. JK触发器

JK触发器中，$G_{1}$,$G_{2}$构成基本触发器，$G_{3}$,$G_{4}$构成触发引导电路，$J,K$为信号输入端，无实际含义

其直接输入为：
$$
{\overline{S_{D} } }={\overline{ {J{\overline{Q^{n} } } }{\cdot}CP} },
{\overline{R_{D} } }={\overline{KQ^{n}{\cdot}CP} }
$$
当$CP=0$时，${\overline{S_{D}}}=1$，${\overline{R_{D}}}=1$，触发器状态维持不变

当$CP=1$时，JK触发器的特征方程为：
$$
\begin{cases}
Q^{n+1}=J{\overline{Q^n} }+{\overline{K} }Q^{n}\
{\overline{S_{D} } }+{\overline{R_{D} } }=
{\overline{ {J{\overline{Q^{n}} } } } }+{\overline{KQ^{n} } }=1
\end{cases}
$$
其中约束条件${\overline{S_{D} } }+{\overline{R_{D} } }=
{\overline{ {J{\overline{Q^{n} } } } } }+{\overline{KQ^{n} } }=1$与$JK$变量的取值无关，即任何条件下均满足约束条件

给出JK触发器的状态转移真值表：

$J$ $K$ $Q^{n+1}$
0 0 $Q^{n}$
0 1 0
1 0 1
1 1 $\overline{Q^{n}}$

JK触发器可以实现钟控RS触发器、D触发器和下文T触发器的所有功能，读者可以尝试改变JK触发器使其实现其他触发器的功能

同时，JK触发器存在“空翻现象”，即当$CP=1$时间较长，$J=K=1$时，由于$Q$和$\overline{Q}$反馈回与非门的时间不同，会导致出现竞争冒险，使得输出$Q$不断翻转，为了解决此问题，引入主从JK触发器
