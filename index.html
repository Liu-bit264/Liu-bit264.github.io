1104第一次测试错题

函数连续性问题

对于连续函数$f(x)$和$g(x)$的和$F(x)=f(x)+g(x)$,$F(x)$必定不连续

因为，假定$F(x)$连续，则有：

$$
g(x)=F(x)-f(x)
$$

则与题意矛盾，必不成立

关于间断点

我们以一个例题来讲明白间断点

点x=1是函数$y=\frac{x-1}{x^2+2x-3}sin\frac{1}{x}$的(    )

A、可去间断点

B、跳跃间断点

C、无穷间断点

D、震荡间断点

我们对原式化简，得到：

$$
y=\frac{1}{x+3}
$$

$(x-1)$上下可消去，则$x=1$为函数的可去间断点

$(x+3)$上下不可消去，且$x\rightarrow3$时$y\neq0$，则$x=-3$为函数的无穷间断点

当$x\rightarrow0$时，$\frac{1}{x}\rightarrow\infin$，此时$sin\frac{1}{x}$震荡，故$x=0$为函数的震荡间断点

数据选择器的使用

地址段多于变量数时

当地址段多于变量数时，多余的地址段接地，多余的数据段接地

地址段小于变量数时

地址端小于变量数时，要对卡诺图进行降维

$A_1$|$A_2$ 0 1
0 1 $A_0$
1 0 ${\bar{A_0}}$

上图便是一个降维的卡诺图，其逻辑函数表达式为${F=\bar{A_1}·\bar{A_2}+A_0·\bar{A_1}·A_2+\bar{A_0}·{A_1}·A_2}$

这是一个三维降二维的例子，下面介绍四维降二维的例子

$B$|$A$ 0 1
0 0 ${\bar{C}D+C\bar{D}}$
1 $C$ $C+D$

原逻辑函数表达式为$F=A\bar{C}D+BC+A\bar{B}C\bar{D}$

注意：降维时选取合适的变量，若变量选取不合适，可能使电路复杂化

完成降维后，对应的数据端按降维卡诺图接到对应的位置
